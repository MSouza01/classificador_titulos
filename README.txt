Devido ao fato de haver somente um campo no qual se basear para a classificação, não foi utilizado um algoritmo específico para a tarefa.

Para executar o classificador, deve estar contido na mesma pasta do código fonte e do arquivo Makefile o arquivo original com os títulos.

Para compilá-lo, deve ser utilizado o comando "make".
Para executá-lo, o comando "make run".
Para executá-lo escrevendo no arquivo de saída "titulos_classificados.tsv", deve ser utilizado o comando "make write_file".
